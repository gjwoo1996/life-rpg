# AI 자기관리 RPG 앱

이 프로젝트는 **자기관리와 게임 시스템을 결합한 데스크탑 애플리케이션**입니다.

사용자는 자신의 삶을 하나의 **RPG 캐릭터 성장 시스템**처럼 관리할 수 있습니다.

현실에서 수행한 활동이 **경험치(XP)**로 변환되고 캐릭터가 성장합니다.

---

# 핵심 컨셉

사용자 = 캐릭터

현실 활동 → 경험치 → 캐릭터 성장

예시

사용자 활동

"JLPT 독해 공부 2시간, 단어 30개 학습"

AI 분석 결과

지능 +4 XP
집중력 +2 XP
규율 +1 XP

총 경험치 = 7

---

# 주요 기능

## 캐릭터 시스템

사용자는 하나의 캐릭터를 생성합니다.

캐릭터 능력치 예시

* 지능
* 집중력
* 규율
* 지식
* 체력

캐릭터는 **도트 캐릭터(pixel character)**로 표현됩니다.

레벨이 올라갈수록 캐릭터 외형이 성장합니다.

---

## 목표 시스템

사용자는 기간이 있는 목표를 설정합니다.

예시

목표

JLPT N2 합격

기간

2026-03-10 ~ 2026-07-01

해당 목표는 특정 능력치를 성장시킵니다.

---

## 활동 기록

사용자는 하루 동안 진행한 활동을 기록합니다.

예시

"JLPT 독해 공부 2시간"

이 기록을 기반으로 AI가 능력치를 계산합니다.

---

## AI 분석 시스템

사용자가 작성한 활동 기록을 **로컬 LLM**이 분석합니다.

AI는 활동을 분석하여 능력치 상승량을 계산합니다.

---

## 캐릭터 성장

캐릭터는 **도트 스프라이트**로 표현됩니다.

레벨에 따라 외형이 변화합니다.

예시

초보 → 훈련생 → 숙련자 → 전문가 → 마스터

---

## 윈도우 프로그램 제어

앱은 Windows 프로그램을 감시할 수 있습니다.

예시

목표

"리그오브레전드 하루 1시간 이하"

동작

프로그램 실행 감지
사용 시간 체크
시간 초과 시 자동 종료

---

## 자동 활동 추적 (추후 기능)

자동으로 사용자의 활동을 기록합니다.

예시

VSCode 사용시간
브라우저 사용시간
게임 플레이 시간

AI가 이를 분석하여 경험치를 계산합니다.

# Windows에서 사용하기

- **exe 받기**: GitHub **Actions** → **Build Windows** 워크플로 실행 후, 완료 시 생성되는 **Artifacts**에서 `life-rpg-windows`를 다운로드한 뒤 압축을 풀고 `life-rpg.exe`를 실행하면 됩니다.
- **사용 방법**: [docs/usage/HOW_TO_USE.md](docs/usage/HOW_TO_USE.md) 참고.
