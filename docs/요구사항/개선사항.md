- 사용자 능력
    - 기본적으로 있는 5개의 능력을 없애고, 사용자가 목표를 추가할때 추가할수있도록,
        - 능력을 추가할때는 기존에 추가된 능력은 select창으로, select창에 없는경우에는 사용자가 input에 직접 기입가능하도록 변경
    - 능력치는 기본적으로 0~100까지의 경험치가 존재하며, 최대 10LV이 존재하게된다.
    - 활동기록을 저장할때는 내용에대한 경험치는 사용자가 기입할수없으며, AI가 분석하여 적절한 경험치를 부여한다.
    - 사용자에 대한 경험치는 가로 퍼센테이지바로 보여주며 focus를 했을때 총 경험치와 현재 경험치를 보여주는 애니메이션을 추가
- 목표
    - 목표는 기존과 동일하나, 캘린더에 표시할 색상을 고를수있도록 추가한다.
- 활동기록
    - 활동기록은 기본적으로, 필수적으로 오늘날짜만 기록이 가능하다.
        - 활동기록을 등록할때는 해당내용을 LLM을 통해 요약하고 저장해둔다
    - 활동기록은 별개의 화면으로 라우팅하며, 총 2dept로 나눈다.
        - 1dept는 캘린더 모양이고, 사용자가 등록한 목표를 색상으로 표시하며, 해당날짜에 요약해둔 활동기록을 보여준다.
        - 2dept는 캘린더의 날짜를 클릭하여 볼수있고, 그날 사용자가 등록한 활동기록과 활동기록을 기반으로 LLM에 분석한 내용을 보여준다.
    - 활동기록의 LLM 사용은 일일단위와 누적단위로 분석하며, 분석시기는 사용자가 활동기록을 등록했을때이다, 일일단위로 분석한건 활동기록에서 캘린더를 클릭했을때 나오는 화면에서 사용할것이며, 누적단위로 분석한건 신규메뉴인 사용자분석 메뉴에서 사용할것이다.
- 신규메뉴
    - 사용자 분석
        - 사용자가 등록한 목표 및 활동기록을 기반으로, LLM을 통해 내용을 분석한다
            - 내용분석은 사용자가 등록한 목표별로 분석하며, 활동기록을 추가할때마다 분석한다.
                - 분석방식은 이전 활동기록과 새로 등록한 활동기록들을 합산하여 분석하고, 이전 활동기록이 많아질경우,
                LLM을 통해 요약된 내용을 DB에 저장하여, 해당 데이터와 신규 활동기록을 이용하여 분석한다.
                - 분석내용은 사용자가 추가한 목표에대해 활동기록들이 제대로 이뤄지고있는지를 확인하고, 학습에 대한 보완점을 사용자에게 안내한다.
        - 일본어에 대한 학습에 대한 LLM(가성비모델선택)을 추가하고, 목표나, 활동기록이 일본어쪽이라면 해당 LLM을 활용하여 분석한다.

위 내용들이 개선사항이며, 계획을 세울때 필요한 내용이 있다면 질문하여 보완해라.
