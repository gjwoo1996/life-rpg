# 업데이트 내역

## 0.0.1

**기준일**: 2025-03-12  
**개요**: GitHub Actions Windows 빌드 워크플로에 버전 자릿수 선택(select) 입력 추가 — 선택한 자릿수만 올리고 나머지는 0으로 설정

---

### 1. GitHub Actions (Build Windows)

- **입력 방식 변경**: 자유 입력 `version` 제거 → **선택(choice)** 입력 `bump_type` 추가
- **옵션**: `major` / `minor` / `patch` 3가지, 기본값 `patch`
- **버전 계산**: `tauri.conf.json`의 현재 버전을 읽어, 선택한 자릿수만 증가·오른쪽 자리는 0
  - `major` → 예: 0.1.0 → 1.0.0
  - `minor` → 예: 0.1.0 → 0.2.0
  - `patch` → 예: 0.1.0 → 0.1.1
- **버전 반영**: 계산된 버전을 `src-tauri/tauri.conf.json`, `src-tauri/Cargo.toml`, `package.json` 세 곳에 동일하게 적용
